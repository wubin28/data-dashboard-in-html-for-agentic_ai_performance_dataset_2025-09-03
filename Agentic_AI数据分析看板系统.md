# Agentic AI数据分析看板系统

## Core Features

- Excel数据读取处理

- 多维度数据分析

- 静态HTML看板生成

- 数据可视化图表

## Tech Stack

{
  "Backend": "Python + pandas + openpyxl + numpy",
  "Frontend": "HTML5 + CSS3 + Chart.js + Bootstrap + JavaScript",
  "Output": "静态HTML文件 + 内联资源"
}

## Design

现代简约浅色调设计，白色主背景配浅蓝色卡片，响应式布局，包含统计卡片、排名列表和交互式图表
